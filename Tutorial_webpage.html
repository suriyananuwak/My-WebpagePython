การเขียนโปรแกรมภาษา Python If...Else

คำอธิบายพื้นฐาน

ใน Python คำสั่ง if...else ใช้ในการตัดสินใจตามเงื่อนไขต่างๆ ภายในโค้ด ดังนี้:

  • if ใช้เพื่อตรวจสอบว่าเงื่อนไขเป็นจริงหรือไม่
  • else ใช้เมื่อเงื่อนไขใน if ไม่เป็นจริง

โครงสร้างพื้นฐานมีดังนี้:

if เงื่อนไข:
    # ทำบางอย่างถ้าเงื่อนไขเป็นจริง
else:
    # ทำบางอย่างถ้าเงื่อนไขไม่เป็นจริง

ตัวอย่างการใช้งาน

สมมติว่าคุณต้องการตรวจสอบว่าตัวเลขเป็นเลขคู่หรือเลขคี่ โค้ดจะเป็นแบบนี้:

number = 10
if number % 2 == 0:
    print("เลขนี้เป็นเลขคู่")
else:
    print("เลขนี้เป็นเลขคี่")

แบบฝึกหัด

ตรวจสอบตัวเลขเป็นบวกหรือลบ:

number = -1
if number > 0:
    print("ตัวเลขเป็นบวก")
else:
    print("ตัวเลขเป็นลบ")

ตรวจสอบเกรดของนักศึกษา:

score = 85
if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
elif score >= 60:
    grade = "D"
else:
    grade = "F"
print("เกรดของนักศึกษาคือ", grade)

ตรวจสอบอายุเพื่อเข้าเรียน:

Age = 18
if age >= 15:

else:
    print("นักศึกษาไม่สามารถเข้าร่วมเรียนได้")
